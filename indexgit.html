Git Push Schritt-für-Schritt-Anleitung im Ubuntu Terminal

1. Datei mit Vim erstellen

$ vim myfile.txt

Drücke i, um in den Einfügemodus zu gelangen, schreibe den gewünschten Inhalt und speichere die Datei mit:

ESC -> :wq -> Enter

2. Git Repository initialisieren (falls noch nicht geschehen)

$ git init

3. Datei zum Staging hinzufügen

$ git add myfile.txt

4. Commit erstellen

$ git commit -m "Added myfile.txt"

5. Remote Repository hinzufügen (falls nicht geschehen)

$ git remote add origin https://github.com/USERNAME/REPOSITORY.git

6. Änderungen ins Remote-Repository pushen

$ git push -u origin main

7. Status überprüfen

$ git status

Falls keine Änderungen mehr vorhanden sind, wurde die Datei erfolgreich gepusht!
