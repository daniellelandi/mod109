How to Upload an HTML File to GitHub

1. Create an HTML File with Vim

Open the terminal and create an HTML file:

vim index.html

Now you are in Vim.

  1. Press i to enter insert mode and write the following HTML code:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First GitHub Page</title>
</head>
<body>
    <h1>Hello, GitHub!</h1>
</body>
</html>

Save and exit Vim by pressing ESC, then :wq and Enter.

2. Prepare Git Repository

If you haven't cloned your repository yet, do:

git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
cd YOUR_REPOSITORY

If you already have a local repository:

cd path/to/your/repository

Or, if you want to create a new local Git repository:

git init

3. Move the HTML File to the Repository

If your file is not inside the repository, move or copy it:

mv ~/index.html .

Or:

cp ~/index.html .

4. Add the File to Git

git add index.html

5. Commit the File

git commit -m "Added index.html"

If you haven't set your Git user details:

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

6. Push the File to GitHub

If your repository is already linked to GitHub:

git push origin main

If your branch is different (e.g., master, dev), replace main accordingly.

If your repository is not linked yet:

git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
git branch -M main
git push -u origin main

7. Verify on GitHub

Go to your GitHub repository and check if index.html is uploaded.

Bonus: Enable GitHub Pages

  1. Go to Settings in your GitHub repository.
  2. Scroll down to Pages.
  3. Select Branch: main and save.
  4. Your site will be available at https://YOUR_USERNAME.github.io/YOUR_REPOSITORY/ after a few minutes.
